<h3>
카테고리 English는 영어 공부한 내용을 기록하기 위한 카테고리 입니다.

이 카테고리에서는 주로 영어 선생님 Tom이 내주는 숙제를 포스팅 할 계획입니다.  

숙제의 내용 구성은 다음과 같습니다.

</h3>

- Q : 톰의 질문 

- A : 나의 대답 (영어)

- ATo_K : 내가 전하려던 의미 (한국어)
